## Dataset overview

LoveDA is a high-resolution (0.3 m/pixel) remote sensing semantic
segmentation dataset covering Nanjing, Changzhou, and Wuhan. Images
are 1024×1024 RGB PNGs; masks are single-channel PNGs with values 0–7.

We use the official Train and Val splits only. The official Test split
has withheld labels (LoveDA benchmark server), so we have no held-out
test set; val doubles as our model-selection set.
